🚀 加密货币交易分析系统 - 快速启动指南

欢迎使用加密货币交易分析系统！这是一个集成了币安和CoinGecko数据的实时分析平台。

📋 第一步：环境配置
1. 确保已安装 Node.js (v14+)
2. 复制 .env.example 为 .env
3. 运行: npm install

🔧 第二步：启动服务
开发模式：npm run dev
生产模式：npm start

🌐 第三步：访问应用
打开浏览器访问: http://localhost:3000

✨ 主要功能
- 实时加密货币价格查询（币安 + CoinGecko）
- 24小时行情数据和涨跌幅排行
- 技术分析指标（RSI、MACD、布林带）
- K线形态识别
- 现货与合约价差分析
- 资金费率监控
- Alpha代币和空投信息
